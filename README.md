# IMDb Clone

## Overview

The IMDb Clone is a web application developed using React, HTML, JavaScript, and Tailwind CSS. It allows users to view trending movies using the IMDb API, manage their own watchlist, and apply filters for a customized viewing experience. The app utilizes session storage to save data, offering a seamless user experience with a clean and modern design.

## Features

- **Trending Movies:** Displays a list of currently trending movies fetched from the IMDb API.
- **Watchlist Management:** Users can add movies to their watchlist, remove them, and view their saved selections.
- **Filtering Options:** Provides functionality to filter movies based on various criteria.
- **Session Storage:** Utilizes session storage to persist data across sessions.
- **Simple and Elegant Design:** Focuses on a user-friendly interface with Tailwind CSS for styling.

## Technologies Used

- React
- HTML
